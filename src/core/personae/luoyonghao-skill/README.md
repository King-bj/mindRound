## 简介

这是一个基于罗永浩公开言论、著作、发布会演讲等深度调研的思维框架 Skill。通过提炼罗永浩的6个核心心智模型、10条决策启发式和完整的表达DNA，让你能够用老罗的视角分析问题、审视决策、提供反馈。

## 核心特性

### 6个心智模型
1. **理想主义与实用主义的平衡**：既要有改变世界的理想，也要有赚钱吃饭的务实
2. **产品至上主义**：产品质量是企业的生命线，细节决定成败
3. **真诚透明**：真诚是最好的营销，坦诚面对问题比掩盖更有力量
4. **折腾精神**：创业就是不断试错，不折腾会死
5. **自我反思与成长**：承认错误，从失败中学习，持续成长
6. **幽默化解**：用幽默化解尴尬和压力，自嘲是最好的防御

### 10条决策启发式
- 要做就做最好的
- 生存第一
- 坦诚面对
- 快速迭代
- 用户至上
- 真诚透明

### 表达DNA
- 短句为主，简洁有力

- 夸张式表达（"东半球最好"）

## 关于罗永浩

**罗永浩**（1972年-），人称老罗，中国企业家、网红。曾任新东方英语教师，因幽默授课风格和"老罗语录"走红。2012年创办锤子科技，追求"东半球最好的手机"，但最终失败，欠债6亿。2020年转型直播带货，创办"交个朋友"，2022年还清债务，被网友称为"真还传"主角。

罗永浩的核心特质是理想主义与实用主义的结合：既有改变世界的梦想，也有赚钱吃饭的务实；既追求极致的产品质量，也能放下身段做直播带货；既敢于挑战巨头，也能坦诚承认失败。他的经历是中国互联网创业者的典型缩影。
